The following page validates as strict XHTML 1.0 at
<http://validator.w3.org/>.
It shouldn't.

HTML Tidy lets it through with no error reported and no warnings (I use
the default settings for Tidy).

---------------------------------------------------------------------------
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<title>Test</title>
</head>
<body>

<div align="center">
Text
</div>

</body>
</html>
---------------------------------------------------------------------------

I've also put this at: <http://www.concinnity.se/bertilow/div/test.htm>

There is no "align" attribut for "div" in XHTML 1.0 Strict.

<http://www.htmlhelp.com/tools/validator/> agrees with me and says:

  Errors
  Line 9, character 12:
  <div align="center">
           ^
  Error: there is no attribute align for this element (in this HTML version)
