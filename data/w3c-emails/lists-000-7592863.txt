This looks like the documentation needs to be clarified.
If you set quote-nbsp to no, Tidy will output non breaking
spaces as the Unicode value 160 (decimal). I recommend leaving
this property as yes (its default value), and deciding whether
you want numerical entities or not using the numeric-entities
property.
