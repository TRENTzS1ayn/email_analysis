This is just a note on compiling tidy on a RedHat 7.1 system,
which we use to mail our webmasters all kinds of errors ...

  1) I would suggest someone writes a README and perhaps a INSTALL file.

  2) make seems to work cleanly. "make all" gives a warning.

  3) The ~/.tidyrc should be used by default on *nix (in platform.h)
