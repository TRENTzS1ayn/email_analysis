The best references are the CSS specifications themselves:

  http://www.w3.org/TR/REC-CSS1
  http://www.w3.org/TR/REC-CSS2

The CSS home page on the W3C site is:

  http://www.w3.org/Style/CSS/

A useful CSS guide in WinHelp format is available for download at:

  http://www.htmlhelp.com/distribution/

and the corresponding online version is at:

  http://www.htmlhelp.com/reference/css/
