Here is an example how a mailer which does not support multipart/choices
might show a multi-language message in the multipart/choices format,
handling it as multipart/mixed:

--- --- ---

Date: Mon, 7 May 2001 11:49:02 +0200
From: Jacob Palme <jpalme@dsv.su.se>
To: jptest@dsv.su.se
Subject: Message in multiple languages
Parts/Attachments:
    1 Shown      0 lines  Text
    2   OK      48 bytes  Text
    3   OK      48 bytes  Text
    4   OK      48 bytes  Text
    5 Shown      2 lines  Text
----------------------------------------

This message is available in multiple languages.

Part 2: English
Part 3: Fran?ais
Part 4: Deutsch

     [ Part 2, Text/plain  48bytes. ]
     [ Not Shown. Use the "V" command to view or save this part. ]


     [ Part 3, Text/plain  48bytes. ]
     [ Not Shown. Use the "V" command to view or save this part. ]


     [ Part 4, Text/plain  48bytes. ]
     [ Not Shown. Use the "V" command to view or save this part. ]


     [ Part 5: "Attached Text" ]
