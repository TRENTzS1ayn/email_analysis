Dear Task Force Members,

If you want to edit parts of the Scenarios document in XML, first
download the newest XML version:
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/Overview.xml

Also, download the following files (you need to do that only once,
they shouldn't change):
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/xmlspec.dtd
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/xmlspec.css
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/xmlspec.ctm
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/xmlspec.tbr
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/xmlspec.xsl

If you want to see the images, download these, too:
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/uswd1.jpg
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/localegui.jpg

And if you want to do the conversion to html yourself, here is
a small .bat file (for Windows only):
http://www.w3.org/International/ws/ws-i18n-scenarios-edit/convert.bat

Regards,    Martin.
