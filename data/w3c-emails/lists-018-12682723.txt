Constructor functions return a specific static type based on the name of
the constructor function. The formatting of the type signature: 

 

pref:TYPE($srcval as xdt:anyAtomicType) as pref:TYPE 

 

 

in section 5.1 should indicate that the return type depends on the name
and is not pref:TYPE (as one could interpret the above).
