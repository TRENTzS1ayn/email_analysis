The definition for fn:sum (15.3.5) states:
"If $srcval contains xs:float or xs:double values, NaN values are discarded."

That means fn:sum() returns a different value than sum() in XPath 1.0
if there are NaNs. If this change is intended it should be mentioned
in appendix B.
