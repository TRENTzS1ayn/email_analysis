The data model spec states that XSLT is a closed language with respect 
to the data model. (1 Introduction)

However, XSLT includes an extension mechanism. 
18.1.3 explicitely allows external objects.

Thus, it looks to me that XSLT is not a closed language.

(Ok, this is an academic comment ...)
