Hello,

I'd like to suggest you the Unified Memory Space Protocol (UMSP),
which is the universal environment for distributed computing.
Integration of UMSP with Web browser allows completely excluding
the embedded program objects from Web pages. At the same time,
any functions of active pages may be realized.

See <http://www.ietf.org/internet-drafts/draft-bogdanov-umsp-web-00.txt>
for details.

Idea of the project is published in RFC3018
<ftp://ftp.rfc-editor.org/in-notes/rfc3018.txt>. 

<http://www.ietf.org/internet-drafts/draft-bogdanov-umsp-rfc3018-update-
00.txt>
contains the specification suitable for industrial realization.

Site of the project is <www.umsp.net>.
