XML Schema part 1 is produced from XML original using augmented xmlspec dtd
using extensively augmented XSLT stylesheet.  Ditto XML Schema part 2, 
same DTD, not quite the same stylesheet.

I use XED to edit the raw XML original, and IE to display a styled
view thereof.  I only run xt (or saxon) to produce an HTML file when
it's time to ship.  I don't know what Paul Biron (the part 2 editor)
edits with.
