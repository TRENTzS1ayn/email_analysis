I don't know if you include application-specific URI schemas - but I thought
I'd mention the following URI schemas missing from the list at
http://www.w3.org/Addressing/schemes.html:


outlook:// - URI to Microsoft Outlook objects

mk:// URI used in microsoft MSDN documentation to link to help topics.

fgjava:// -
http://www.macromedia.com/support/generator/ts/documents/datasource_java.htm

fgjdbc:// -
http://www.macromedia.com/support/generator/ts/documents/jdbc_odbc.htm

fpftp:// -
http://www.macromedia.com/support/generator/ts/documents/datasource_ftp.htm

Last three are used by macromedia Generator application to specify
datasources for flash movies
