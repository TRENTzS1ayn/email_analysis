Chris,

As an Apache admin, if you want to enable WebDAV on your server, you'll
probably be interested in either:

* mod_dav, an Apache module which provides WebDAV services.
  http://www.webdav.org/mod_dav/

* DAV4J, a Java servlet which provides WebDAV support, and works with Apache
  http://www.alphaworks.ibm.com/tech/DAV4J
