Dear all,

We are pleased to announce the latest release of the ICS-VRP V 2.5
available for download.

New Features:

1. VRP Performance Enhancement
   ~ 20 % less memory and ~ 50 % less time

2. Full Support of XML Schema Datatypes
   Integration of the Sun" XML Datatypes Library
   http://wwws.sun.com/software/xml/developers/xsdlib/

3. Support of the latest RDF/XML Syntax Specification
   proposed by W3C Working Draft 23 January 2003
   http://www.w3.org/TR/2003/WD-rdf-syntax-grammar-20030123/

4. Additional VRP Statistics

   Direct and recursive ancestors and descendants
   distribution for both class and property subsumption
   hierarchies

5. SVG (1.0.) Representation of RDF/S Schema and Resource Descriptions
   Navigate to class and property subsumption hierarchies
   Lookup declared or inferred class and property descriptions
   Discover related resource description graphs
   Filter description graphs using namespaces

We are looking forward to receiving your comments and suggestions.
