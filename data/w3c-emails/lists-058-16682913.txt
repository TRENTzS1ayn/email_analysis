Technique for Checkpoint 4.1

Offer the user 3 levels of checking:
1. Check for Triple-A Conformance
2. Check for Double-A Compliance
3. Check for Single-A Compliance

The author could also be offered the option (if available) to 
save the choice as the default checking mode, or choose to be 
prompted each time.  If such a methodology is implemented, then 
"Check for Triple-A Conformance" should be the default choice. 
(Note: the choices listed above appear a specific order, so as to 
conform with Checkpoint 6.2)
