I'm taking a vacation day to be with my kids, who are out of school, so I
will miss today's conference call. However, I did get a start on my action
item concerning user agent capabilities.

I haven't gotten to the techniques yet, but here's what I came up with for
2.0 checkpoints. Comments, corrections, etc. welcome. I'm unfortunately not
going to be able to attend the face-to-face, but I will try to get this list
updated by the time of the meeting.

Dick Brown, Microsoft

Action item:
Work through 2.0 and latest techniques to identify user agent capabilities
which are assumed by the checkpoints and techniques.


1.1 Provide a text equivalent for all non-text content.
Capabilities assumed:
ALT; LONGDESC; TITLE
 
1.2 Synchronize text equivalents with multimedia presentations.
Capabilities assumed:
Ability to display multimedia or launch separate multimedia appliction(s).
 
1.3 Synchronize a description of the essential visual information in
multimedia presentations. 
Capabilities assumed:
Ability to display multimedia or launch separate multimedia appliction(s).
 
1.4 Use markup or a data model to provide the logical structure of content. 
Capabilities assumed:
HTML 4.0 (?)
  
1.5 Separate content and structure from presentation. 
Capabilities assumed:
Style sheets

2.1 Provide consistent interaction behaviors and navigation mechanisms.
Capabilities assumed:
ACCESSKEY

3.2 Emphasize structure through presentation, positioning, and labels.
Capabilities assumed:
TITLE;  LABEL for=

3.4 Use multimedia to illustrate concepts.
Capabilities assumed:
Ability to display multimedia or launch separate multimedia appliction(s).

3.6 Define key terms, abbreviations, acronyms, and specialized language.
Capabilities assumed:
ABBR
