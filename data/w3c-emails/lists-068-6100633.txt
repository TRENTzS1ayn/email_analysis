The Evaluation And Report Language is an RDF based framework for
recording, transferring and processing data about automatic and manual
evaluations/tests of resources or tools.

The purpose of EARL is to provide a framework for generic evaluation
description formats that can be used in the new QA effort at W3C, and
as part of the WAI ERT - Evaluation and Repair - Working group, in
which it it being developed.

Home page at
