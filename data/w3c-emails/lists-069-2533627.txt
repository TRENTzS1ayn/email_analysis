Abstract
This document describes techniques for authoring accessible Scripts
including Javascript. This is the first Working Draft associated with the
latest Web Content Accessibility Guidelines 2.0 Working Draft.

Status of this document
This document is prepared by the Web Content Accessibility Guidelines
Working Group (WCAG WG) to show how a Script Techniques for WCAG 2.0 with
more easily testable script-specific checkpoints might read. This draft is
not based on consensus of the WCAG Working Group nor has it gone through W3C
process.
