A question was asked last week as to how to use frontpage to make client side image maps. here is the response from my programmer.

<start snip>
The code can be put into FrontPage98 using the Insert > FrontPage
Component... > Insert HTML then pushing OK.  A text box will pop up to place
the code in.  Unfortunately, if the client-side image map with alt tags is
done through the normal FrontPage98 route all of the ALT tags will be
stripped out.

<end snip>

If this response is not clear, please reply and we will ask hime to be more specific. I would have asked but he has gone for the day.
