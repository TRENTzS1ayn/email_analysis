WAI IG CALL FOR REVIEW: "User Agent Accessibility Guidelines" Working Draft
          
The User Agent Guidelines Working Group (UAWG) has prepared a Working Draft
which they would like the WAI Interest Group to review before publishing it
as an updated version on the W3C Technical Reports page. Please review the
draft, dated March 9, 1999
<http://www.w3.org/WAI/UA/WD-WAI-USERAGENT-19990309>. Comments are due by
Tuesday, March 16, 1999. Details follow.
         
TIMELINE FOR WAI IG REVIEW:
Please send comments by close of business US Eastern Standard Time,
Tuesday, March 16, 1999, at the latest. Earlier comments are greatly
appreciated. We will do our best to incorporate comments received by this
deadline.
          
WHERE TO SEND COMMENTS:
Send comments to the <w3c-wai-ua@w3.org> list. The UAWG mailing list is
archived from the UA home page <http://www.w3.org/WAI/UA> if you would like
to view other people's comments.
     
CHANGES:
The list of changes to the document is available at:
<http://www.w3.org/WAI/UA/wai-ua-wd-changes>. Please note that this draft
will replace the November, 12, 1999 version and includes substantial
changes from that version.

REVIEW QUESTIONS:
Please address the following questions in reviewing this draft. Other
feedback is also welcome.

Specific questions for this draft:
1. Are there additional Guidelines, Checkpoints or Techniques which need to
be included?
2. Are the priorities of Checkpoints appropriate as indicated (priority
levels are defined within the document)?
3. Are there Checkpoints which should be Techniques, or Techniques which
should be Checkpoints?

General Questions for the document:
4. Is the document understandable and clear?
5  Does the format/structure of the document work well?

THANK YOU:
To the Chair, Editors, and User Agent Working Group members.

QUESTION?
If you have any questions on this process, please contact the Chair of the
group, Jon Gunderson <jongund@staff.uiuc.edu>, the W3C Staff Contact, Ian
Jacobs <ij@w3.org>, or WAI Domain Leader Judy Brewer <jbrewer@w3.org>.
