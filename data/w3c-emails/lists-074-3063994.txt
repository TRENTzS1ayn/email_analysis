English:
Your pages are excellent. But i would like to make a small correction. It
is true that many HTML authoring programs cannot be used to produce
accessible pages, but it is not true of all such programs. There are some
which show the HTML code itself, and there are some which have been
designed with accessibility as an important goal.

Also, the W3C has an activity - the WAI Authoring Tools Working Group,
http://www.w3.org/WAI/AU - which is working on these problems. I hope and
believe that it will be easy, before the year 2000, for somebody who does
not know HTML to use such a program and produce things which are
accessible. It is already possible, but it is a long way from universal.
