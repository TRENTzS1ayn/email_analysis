At 10:05 AM 11/16/1999 , Claude Sweet wrote:
>Might it be possible to actually set a command on the browser used by
>the disabled user so the web site would send a query and automatically
>send the user to the proper page maximized for the user's specific

You mean maximized for the user's specific _ability_. :)

This is exactly what CC/PP -- Composite Capabilities/Preferences
Profiles -- will do for us in the future. :)  CC/PP is under development
at the W3C in the CC/PP working group, and should have some solid
foundations set next year.  For a longer explanation (on a really
basic, boring page!) see:

      http://www.ccpp.org/

Or the W3C site:

      http://www.w3.org/Mobile/CCPP/
