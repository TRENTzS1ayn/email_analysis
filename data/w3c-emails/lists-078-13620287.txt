New in 4.8

A) AccVerify(r) Web Test Feature
Configure and use this Web Based accessibility Testing Service as part
of your AccMonitor solution. Users can test from any desktop or server
(linux, Macintosh, Solaris, etc...) for that matter where a Web Browser
can open a HTML Page and connect to the Web Testing Service. 

More Importantly the Visual Interviews can be completed, usage can be
controlled and all information on runs can be logged and stored.
Supported Standards are Section 508, WCAG 1-3, CLF and All Standards
Derived from the same.

B) AccMonitor Updated Team Services
Cluster your AccMonitor and AccVerify Desktop Installations together via
data collection Webs, allowing for the sharing of resources and
centralized or clustered reporting. Combinations of Data can be
completed between

- One to many AccMonitor Installations
Includes .Net Integrations
- AccVerify/AccRepair to AccMonitor Installations
Includes Mercury Interactive TestDirector, WinRunner, QuickTest
Professional Data Results
- AccVerify/AccRepair for FrontPage to AccMonitor Installations



C) Reporting Enhancements
Integrated Schedule Data Collection and Roll-ups for Combined Enterprise
Reporting across one of Clusters of Servers and Desktops

D) Enhancements to Interface
The AccMonitor Interface was updated with multiple usability upgrades.
