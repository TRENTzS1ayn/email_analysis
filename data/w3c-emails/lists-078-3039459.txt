Hello All-

I am writing to let you know that HiSoftware is announcing the release of a
new component of its AccVerify product line that will enable complete
automated and custom visual usability testing. This new component will be a
no charge add-on for user's of HiSoftware's AccVerify/AccRepair,
AccVerify/AccRepair for FrontPage, and AccVerify SE for FrontPage.

This new add-on system will augment the existing automated usability testing
available in AccVerify providing a visual verification and tracking system
for all usability testing. The Usability Visual Test Manager will enable
users to quickly and interactively create custom tests and test groups as
well as providing them with the ability to create their own tests and to mix
and match question files.


Features include:

Question/Answer Wizard powered by XML
Question Manager and Editor provided for EASY Creation of Usability Tests
with NO Knowledge of XML required
Add or Create corporate Standards
Allows Boolean, Multiple Choice Choose from Answer Group and more for Answer
and Question formation
Allows or Blocks Comments
Output to XML or Database
Ships with a Complete Usability Test group covering usability areas:
Clarity, Consistency, Content Structure, Site Feedback, Help & Site Help,
Navigation Systems, and User control.
