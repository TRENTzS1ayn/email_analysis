Or, if the password protected site is using standard htaccess password
protection then the HTML Help Validator can be used:

  To validate password-protected documents, use a URL of the form
  "http://user:pass@www.yourdomain.com/foo/bar.html". (While this is not
  a technically valid URL, it is accepted by the WDG HTML Validator for
  convenience.)

  http://htmlhelp.com/tools/validator/tips.html#password
