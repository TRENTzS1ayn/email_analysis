ATutor 1.3.1 has been released. 

ATutor is an Open Source Web-based Learning Content Management System (LCMS) designed with accessibility and adaptability in mind.


New in this Release
Automated Install/Upgrade Script: Installing and upgrading has been automated through a step-by-step interface that walks users through the process.

Table Prefixes: The table prefix option allows ATutor to share an existing MySQL database with other applications and tables, and avoid conflicts between like named tables.

Administrator & Instructor Accounts: Customizable administrator and initial instructor usernames and passwords are now created during the install process.

Direct Administrator Login: No longer have to be logged in as a user to login as an administrator.

Languages: Improved right-to-left language support. In addition to English, current languages include: French, Spanish, and Italian. Other language currently being translated include: Arabic, Farsi, Chinese, Chinese Simplified, Dutch, Danish, German, and Croatian.

Viewing Courses: Administrators can now directly enter and view courses through the course manager.

XML Parser Update: The PEAR XML_HTMLSax library, used to parse HTML content for content packaging, has been updated to the latest version.

Additions and Enhancements: Lots of minor fixes, additions, and enhancements.
