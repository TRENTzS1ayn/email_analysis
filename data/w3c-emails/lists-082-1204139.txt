New issues

Issue 19. Physically Impaired Navigation
Issue raised by: Bryan Campbell , Jennifer Angelo
Date: July 9th, 1998 and June 29th, 1998
Issues:
1. Changing the size of toolbar controls and page anchors to accommodate poor
fine motor skills 
2. Clustering keyboard navigation controls in close proximity for improved
navigation performance 

Issue 20. Dynamic Visual Effects in Elements or Applets
Issue raised by: Mimi Cummins
Date: July 2nd, 1998
Issues:
1. Blinking text rate causing seizures 
2. People with visual impairments or learning disabilities not being able to
read moving or blinking text 
3. How will Active-X controls, Java Applications and other OBJECTS and APPLETS
know to suspend dynamic visual effect 

Solution Options:
1. Operating system flag that suspends dynamic visual effects like BLINK and
MARQEE 
2. Browser option that that suspends dynamic visual effects 
3. Tag rendering specifications for non-HTML 4.0 flags like MARQEE and BLINK 

Issue: 21. Semantic Markup of Links for UA Direct Navigation
Issue raised by: Jon Gunderson
Date: June 30th, 1998
Issues:
1. Improve direct navigation capability by providing semantic mark up of
Anchors

Solution Options:
2.  Use the RELative attribute of the Anchor tag to indicate the purpose of an
Anchor. User agents could create list of different types of links to
present to
the user for direct navigation 

Issue 22. Semantic Markup of Tables for UA Rendering
Issue raised by: Jon Gunderson
Date: June 30th, 1998
Issues:
1. Adjust rendering of table information based on the purpose of the table
