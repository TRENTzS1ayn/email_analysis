Part 2: INSTALLATION NOTES

1. first application window (title: "Opera 3.61 
(Win9x/NT 4.0) Installation")

1a. only window title and item with current focus (Next 
button) voiced. needed to use screen review to read the 
contents of the application window.

2. second application window (title: "Opera 3.61 
(Win9x/NT 4.0) Installation")

2a. only window title and item with current focus (Next 
button) voiced. needed to use screen review to read the 
contents of the application window--a header and the 
edit box containing the licence agreement.

3. third application window (title: "Opera 3.61 
(Win9x/NT 4.0) Installation")

3a. only window title and item with current focus (Next 
button) voiced. needed to use screen review to read the 
contents of the application window.

4. fourth application window -- nothing spoken (header: 
Select Group)

4a. using the JFW read-box-in-tab-order keystroke (INSERT+B)
contents of application window spoken clearly and in proper 
order.

5. fifth application window -- nothing spoken (header: Ready 
to Install)

5a. using the JFW read-box-in-tab-order keystroke (INSERT+B)
contents of application window spoken clearly and in proper 
order.

6. sixth application window -- (title: "Opera 3.61 (Win9x/NT 
4.0) Installation"; header: Installation Complete!)

6a. only window title and item with current focus (Finish  
button) voiced. needed to use screen review to read the 
contents of the application window.

7. OVERALL: Although the application windows did not 
automatically voice, installation was easily accomplished 
using commonly used screen review and navigation 
commands
