I invite you to review the W3C User Agent Accessibility Guidelines[1].  The 
document has been released for its third last call review.  Changes have 
been made to the document based on the previous last call review.  The 
group does not anticipate many changes to this document before moving to 
the next stage in the W3C process.  Please review the document for 
clarification of requirements or suggestions on how to improved 
usability.  Any requests for new requirements will be archived on a issues 
list for the next version of the guidelines.  Changes in Priorities of 
checkpoints will only be considered if there is substantial new information 
included in the review for the group to consider.

Please read the last call announcement [2] carefully before reviewing the 
document and sending in comments.

Thank again for your help and time in developing the User Agent 
Accessibility Guidelines,
