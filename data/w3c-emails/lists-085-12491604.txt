This is a proposed revision of checkpoint 4.2 based on discussions on the 
19 July teleconference[1].

<CURRENT>
Allow the user to choose from among the full range of font families offered 
by the conventional font selection mechanism of the operating environment.
</CURRENT>

<NEW>
Allow the user to choose from among the full range of font families offered 
by the conventional font dialog box resource supported by the operating 
environment. If the operating environment does not offer a conventional 
font dialog box resource, allow the user to choose any font family 
supported by the conventional text drawing APIs of the operating environment.
</NEW>
