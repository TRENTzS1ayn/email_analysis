January 5, 2003
Making the Web More Accessible

By  Darryl K. Taft

The end of the year, which typically signals slowdown in the technology
industry, instead saw a flurry of activity in the Web applications and
Web services standards space.
Closing out the year, the World Wide Web Consortium (W3C) last month
released the User Agent Accessibility Guidelines (UAAG) 1.0 to help
developers make Web browsers and multimedia players accessible to
disabled individuals.
