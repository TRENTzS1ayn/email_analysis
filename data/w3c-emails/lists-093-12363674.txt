Hussein wrote:

> how to centre text in Amaya?

There are two ways:

  - Associate a style attribute that contains a CSS property
    "text-align: center"
    First select the paragraph or block to be centered and use the
    Style menu.  Entry Format displays a dialog box with an Align
    menu.

  - Use the HTML attribute align
    First select the paragraph or block to be centered and use the
    Attribute menu.  Select entry align and choose center.
