Hi William,
Yes you should download the latest release.

For those who cannot run amaya, please check that the file thot.ini 
(located in Thot\config directory) is correct. Especially the character
'\' is used a separator instead of '/'.

A typical thot.ini file should look like the following description:

------------


[thot_lib]
THOTSYSSCH=$(THOTDIR)\amaya

[amaya.exe]
THOTSYSSCH=$(THOTDIR)\amaya
LANG=en_US
HOME_PAGE=http://www.w3.org/Amaya
