[Announcement] Amaya 1.2a public release

    The Amaya team is proud to announce the availability of the 1.2a release
    of Amaya editor/browser.

    This release includes minor changes and fixes a set of bugs in Unix
    and Windows versions. The same distribution is available for Windows'95
    and Windows NT and its installation is now automatic.

    The binary is available at:
            http://www.w3.org/Amaya/User/BinDist.html

    The source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

    This release includes:

* automatic save when Amaya crashes (Unix)
* fixing of memory leaks (Windows)
* corrected bugs concerning browsing, images and editing
See http://www.w3.org/Amaya/User/New.html for more details
