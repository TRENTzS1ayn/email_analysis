Amaya 2.1 does not display hex character entities properly.  For example
when a document contains &#x5e; I should see a "hat/caret/circumflex" (i.e.,
shift 6 on English keyboards).  IE and Navigator display the appropriate
character Amaya displays "xx5e;".

In addition there doesn't seem to be any way to enter character entities in
Amaya.  Inserting &#x5e; and saving the html to a file results in &amp;#x5e;
