The Amaya 3.1 release is now available for both Unix and Windows platforms.

  Binaries (Solaris, Linux, Windows NT/95/98) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

This version fixes a lot of bugs, but it also includes new features:
- MathML: attributes lquote and rquote on element ms are now taken into account in the formatted view.
- Lines are numbered in all text and source files
- WAI: It's now possible to completely create a link from the keyboard
- The "Check spelling" can be now applied to text files and source views

For more details please read the page
    http://www.w3.org/Amaya/User/New.html
