The Amaya team is proud to announce the Amaya 6.1 release.

  Binaries (Solaris, Linux, Windows NT/2000/XP /9x/ME) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

Debian packages and rpm are also available.

This is an update of the 6.0 release that fixes a number of bugs.

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/

Thanks to all the Amaya users for their feedback, suggestions, and 
contributions so far.
