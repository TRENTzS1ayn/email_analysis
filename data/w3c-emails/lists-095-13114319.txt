> Questions:
> 1. Can an "xlink:href" inside a "use" refer to another document according to
> the SVG spec?

Yes. Refer to http://www.w3.org/TR/SVG/struct.html#Head

> 2. If so, is my syntax below a valid representation?
> 3. Also if so, does Amaya currently support this feature?

No. It currently supports only local URI references with the use
element, i.e. the <use> element and the element it references
must be in the same file. There are plans for full support of
URIs in the use element, including non-local URI references.
