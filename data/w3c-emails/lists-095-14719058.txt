The Amaya team is proud to announce the Amaya 6.2 release.

  Binaries (Solaris, Linux, Windows NT/2000/XP /9x/ME) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

  Debian packages and rpm are available at:
            http://www.w3.org/Amaya/User/BinDist.html.

This is an internationalized version, based on the unicode support, that 
broadens the number of international document supported by Amaya. Morover, the 
number of encodings now includes Windows-1250 to Windows-1257, iso-8859-1 to 
iso-8859-15, koi8-r, iso-2022-jp, shift-jis, and utf-8.

This release includes mainly bug fixes but also some new features:
- Easier install on Windows platforms:
  You can now select between atypical (complete), compact, or custom 
installation of Amaya.
- A German version of the Amaya documentation (thanks to Rudolf Troeller).
- Extensions of the CSS parser
- Progresses with the SVG support
- Support of STIX fonts in the OpenGL version
- Association of different icons to different annotation types.

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/

Thanks to all the Amaya users for their feedback, suggestions, and 
contributions so far.
