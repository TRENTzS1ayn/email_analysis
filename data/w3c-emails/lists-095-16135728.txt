The Amaya team is proud to announce the Amaya 6.3 release.

  Binaries (Solaris, Linux, Windows NT/2000/XP /9x/ME) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

Debian packages and rpm are also available.

This is an update of the 6.2 release that fixes a number of bugs.
New features concern:
- New preference options.
- Possibility to control whether images are loaded or not.
- Implementation of the SVG element tref and the attribute baseline-shift.
- Implementation of MathML attributes framespacing, rowlines, columnlines,
  rowspacing, columnspacing for element mtable.
- Support of new Unicode characters.
- Support for annotations described with DC 1.1.

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/

Thanks to all the Amaya users for their feedback, suggestions, and 
contributions so far.
