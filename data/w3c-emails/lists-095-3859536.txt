The Amaya team is proud to announce the Amaya 4.3.1 release.

  Binaries (Solaris, Linux, Windows NT/2000 /95/98) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

This release mainly provides bug fixes, but also some new features:
- Character entities are now handled correctly
- It's now possible to reparse ISO-latin-1 XHTML documents which are
  missing the charset.
- The user can now create SVG paths
- It's now possible to move an existing annotation inside a document
- Digest authentication now works better with HTTP/1.1 pipelined requests

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/
