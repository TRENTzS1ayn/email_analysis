The Amaya team is proud to announce the Amaya 4.3.2 release.

  Binaries (Solaris, Linux, Windows NT/2000 /95/98) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

This release restores the support of some important MathML entities like 
&ThickSpace;
and the handling of the entity &amp; within attribute values.
It includes other bug fixes too.

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/
