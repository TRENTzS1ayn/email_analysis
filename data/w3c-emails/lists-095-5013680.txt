The Amaya team is proud to announce the Amaya 5.0 release.

  Binaries (Solaris 8, Linux, Windows NT/2000 /95/98) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

This release includes the following new features:

- New accessibility features (keyboard navigation and map element).
- Enhacement of the SVG support, like reference of SVG files from HTML 
  (img, object) and SVG (image) elements.
- Right clicking on a link will now open a new window.
- Better displaying of PNG images and better support of colors in general.
- The CSS parser gives now some feedback: errors and warnings. They are 
  displayed by the command Views/Show parsing errors.
- Support of XHTML 1.1, including both simple and complex Ruby Annotations.
- It's now possible not only to browse image files, but also to save them.
- The user can change the Charset and MIME type of a document with the Save As menu.
- Much progress in the GTK port of Amaya (first Alpha release now available).

and other bug fixes.

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/
