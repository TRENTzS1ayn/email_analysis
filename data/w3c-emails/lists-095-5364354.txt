The Amaya team is proud to announce the Amaya 5.1 release.

  Binaries (Solaris 8, Linux, Windows NT/2000 /95/98, Linux GTK) are available 
at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

This release fixes a set of reported bugs in the previous release and includes 
small changes:

- The button bar uses the flat style.
- There is a Portuguese translation of Amaya dialogues contributed by Ricardo 
Maciel.
- An index on the online documentation is available.
  This index was developed by WinWriters (http://www.winwriters.com).

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/
