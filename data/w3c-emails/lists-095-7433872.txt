The Amaya team is proud to announce the Amaya 5.2 release.

  Binaries (Solaris 8, Linux-Motif, Linux-GTK, Windows NT/2000 /95/98) are 
available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

This release includes the following new features:

- Support of generic-xml documents (in browser mode only)
- Interpretation of the embed element when it refers to an SVG 
  or a MathML document for compatibility reasons.
- In the Windows version a new item "Export CR/LF" of the menu 
Preferences/General
  allows users to generate a DOS compatible file format.
- Handling of the HTTP Content-Location header when publishing.
- Support of the Alt-Gr Windows compose key.
- By popular demand, the French and English dictionaries were updated and are 
now included
  in the binary distribution.

and a lot of enhancements and bug fixes.

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/
