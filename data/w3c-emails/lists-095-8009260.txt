Well, to demote an H1 you can select it (the element, using esc/F2 depending
on platform) and then select paragraph from teh types menu. (I don't know why
it doesn't work with the keyboard shortcut, but it didn't ever do so for me)

For a list you can select it and then select Edit->transformations->remove
one level

If you only want to do this for one item, you should seperate it from the
list first (return, return, return before and after the item to seperate from
the middle). There is also an option to merge two lists (must be same type
and both selected) in the transformations menu.
