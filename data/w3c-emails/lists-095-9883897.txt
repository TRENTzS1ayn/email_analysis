The Amaya team is proud to announce the Amaya 6.0 release.

  Binaries (Solaris, Linux, Windows NT/2000/XP /9x/ME) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

Debian packages and rpm will be made available later this week.

This is an internationalized version, based on the unicode support, that 
broadens the number of international document supported by Amaya. Morover, the 
number of encodings now includes Windows-1250 to Windows-1257, iso-8859-1 to 
iso-8859-15, koi8-r, and utf-8. The HTML element bdo and dir attribute as well 
as the CSS property direction are also supported.

This release also includes a set of new features:
- support of new MIME types application/mathml+xml, application/xhtml+xml, 
  application/svg+xml, and application/xml
- better editing and rendering of MathML
- SVG elliptical arcs
- Annotation bodies can now be of type plain text, SVG, and MathML
- CSS cascade is now properly supported as the !important
- Attributes and attribute values are now supported in CSS selectors
- Support of CSS properties display: block and display: inline
- Support for all X11 color keywords as specified in the CSS3 color module
- New browsing/editing functions are also available

Much progress has been done in the GTK version and a set of reported bugs are 
now fixed.
Moreover, a first OpenGL version is now released on both Linux and Windows 
platforms.

For more details, please read the page
    http://www.w3.org/Amaya/User/New.html

If you're interested by annotations, visit the Annotea project home page:
    http://www.w3.org/2001/Annotea/

Thanks to all the Amaya users for their feedback, suggestions, and 
contributions so far.
