The Amaya team is proud to announce the Amaya 7.0 release.

  Binaries (Solaris, Linux, Windows NT/2000/XP /9x/ME) are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

  Debian packages and rpm are available at:
            http://www.w3.org/Amaya/User/BinDist.html.

This release includes mainly bug fixes but also new important features:
  - Some changes in the user interface (the history of last open URIs, reorganization of menus)
  - A better support of XML documents: styling, some editing features
  - A complete Open GL version that provides a better SVG rendering
  - Progress on CSS (possibility to test edits in a CSS style sheets without saving it)
  - The possibility to edit SVG animation even if the current version doesn't play animation.
  - A library manager that allows authors to get predefined SVG components and to create his own
    set  of reusable SVG components.
  - Migration to the Raptor RDF parser.

More details are available at
http://www.w3.org/Amaya/User/New.html
