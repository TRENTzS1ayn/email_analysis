Hi,

If I write "classname" in the "Use as class" dialog, a ".classname" is
created
in the <head> of the document.
If I write "p.classname" in the "Use as class" dialog, a ".p.classname" is
created in the <head> of the document.
Is it possible to create a "p.classname" class through the Style > Create
rule
menu?

By the way, a .p.classname is not applied by Amaya (neither Mozilla nor IE),
but W3C CSS validation service validates it as valid. This is an example:

<head>
  <meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1" />
  <title>No title</title>
  <style type="text/css">
.p.test {color: white; background-color: black}</style>
  <meta name="generator" content="amaya 8.1b, see http://www.w3.org/Amaya/"
/>
</head>

<body>
<p class="p.test">Hello, world</p>
</body>
