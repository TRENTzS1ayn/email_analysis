The Amaya team is proud to announce the Amaya 8.2 release.

Thanks to contributors that tested the pre-release.
I remember new features included in that release:
 - A new selection.
 - A CSS debug command: "Show applied style" shows CSS rules applied to the current selected element. 
 - Generation of backup files.
 - A separate control of loaded objects and loaded images.
 - Detection of a unchanged document after undo.
 - Coloration of the structure view and the source view.
 - Parsing (but not support) of all SVG 1.0 elements and attributes.
 - Support of CSS selectors "*", "[att|=val]", pseudo-class :lang.
 - Parsing (but not support) of all CSS 2 properties.
 - New HTML transformations
 - Annotea upgrade to Redland 0.9.14
 
  Binaries (Linux, Windows NT/2000/XP /9x/ME), Debian and RPM 
  packages are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

  For more details about this release see:
           http://www.w3.org/Amaya/User/New.html
