CSS 2 defines 28 color names related to the operating system's graphic
environment (ActiveBorder, ActiveCaption, AppWorkspace, Background, etc.).
These color names will be deprecated in the future CSS3 Color module, but
they can be used today.

Bugs in Amaya 8.2:
* Amaya does not display these colors.
* Amaya does not recognise these colors and generates an "Invalid color
   value" parsing error message.
* Amaya generates an strange "CSS property ignored "kground"parsing error
   message when the Background color is used.
