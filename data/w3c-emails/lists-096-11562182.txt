Unfortunately the latest release on Nov 13, 2003 does not render 
my educational site very well.  This site has many tutorials for
webpage design  and Java  and JavaScript.

Many code fragments use the preserve tag with font weight bold!
The monospace bold does not have correct spacing for the space character
as exhibited by the included monospace.htm test.

Also in my main style i use a tartan background for h2 headers.  my tutorials
have a cascading style that reverts the background back to a simple color.
Unfortunately Amaya does not acknowledge this alteration.
See backurl.htm and tartan.gif  for a demo of this error.

There are still many problems with table widths calculation as well.  I will
forward simple tests as they are prepared. Hopefully these can be used
to isolate the problems in simple element rendering. --
