On page <URL:"http://www.w3.org/Amaya/User/Changelog.html"> new features in 
the CVS version of Amaya since the last release (Amaya 8.2) are listed:

"Generate the last saving date on request: the format is 
[dollar]Date:[dollar] like in the following example $Date: 2003/11/26 
17:32:03 $ "

Either the date and time format specified in the locale of the system in 
question or the ISO 8601 format, i.e. 2003-11-26 17:32:03, should be used.

The ISO 8601 format should also be used at the foot of the page referred to 
above.
