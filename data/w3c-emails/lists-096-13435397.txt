The Amaya team is proud to announce the Amaya 8.2+ Snapshot.

  Binaries (Linux, Windows NT/2000/XP), Debian and RPM 
  packages are available at:
            http://www.w3.org/Amaya/User/BinDist.html

This snapshot fixes a set of annoying bugs and includes new features.
- Generation the date (on request)
- New transformations: <p> to <pre> and <pre> to <p>.
- New set of shortcuts:
   Ctrl t Ctrl ... for table editing
   Ctrl o Ctrl ... for forms
   Ctrl i Ctrl ... for information and style editing
   Ctrl m Ctrl ... for MathML
   Ctrl h Ctrl ..
- A new table editor

For more details see 
    http://www.w3.org/Amaya/User/Changelog.html
