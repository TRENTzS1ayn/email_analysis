The Amaya team is proud to announce the Amaya 8.3 pre-release.

  Binaries (Linux, Windows NT/2000/XP), Debian and RPM 
  packages are available at:
            http://www.w3.org/Amaya/User/BinDist.html

This release fixes a set of bugs and includes new features:
- A powerful table editor
- New transformations rules
- New dialogs (for table edits, the map element, character encodings, and document types)
- New set of shortcuts:
  Ctrl t Ctrl ... for table editing
  Ctrl o Ctrl ... for forms and open functions
  Ctrl l Ctrl ... for links
  Ctrl i Ctrl ... for information and style editing
  Ctrl m Ctrl ... for MathML
  Ctrl h Ctrl ... for other HTML elements
  Ctrl n Ctrl ... for annotations and new functions
  Ctrl v Ctrl ... for views 
 - Generation the date (on request)
 
For more details see 
    http://www.w3.org/Amaya/User/Changelog.html
