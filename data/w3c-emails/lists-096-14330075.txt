The Amaya team is proud to announce the Amaya 8.3 release.

  Binaries (Linux, Windows NT/2000/XP), Debian and RPM 
  packages are available at:
            http://www.w3.org/Amaya/User/BinDist.html

This release fixes a set of bugs and includes new features:
  - New functions to:
    o select, insert, copy/paste, delete of  table rows and columns,
    o idem for MathML matrices,
    o create of map elements, 
    o change the character encoding
    o change the document type
    o control the content type
  - New set  of transformation rules
  - New set of shortcuts:
  - Automatic generation of the date (on request)
  - Automatic save of windows geometry
  - Improvement of text and image editing
  - Improvement of table formatting
 
For more details see 
    http://www.w3.org/Amaya/User/Changelog.html
