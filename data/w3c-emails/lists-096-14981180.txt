Red Hat Linux 3.0.3 is the latest distibution from Red Hat, released in   
March, 1996. Red Hat Linux represents a quantum leap in Linux usability.   
It is the easiest Linux distribution to install and maintain. Beginners   
will find Red Hat 3.0.3 suits their needs, while experienced users will   
find it to be a very powerful development tool. Now includes Metro-X   
Enhanced X Server! New version of RPM! Two CD set ships with the Red Hat   
User's Guide.

Included on the CDs:

 Red Hat Linux 3.0.3 distribution  Live File System CD (run Linux   
directly from your CD)  Metro-X Enhanced X Server



Red Hat Linux 3.0.3 comes with hundreds of compiled and ready to install   
additional packages. A full package list in included at the end of this   
description.

Features include:

 Fully ELF based  entirely upgradeable (no reinstallation necessary)   
 compliant with version 1.2 of the Linux File System Standard  Red Hat   
Motif and Linux Library (add-on products) will seamlessly install  RPM   
2.0: new version of RPM provides many new features and faster performance   
 Metro-X Enhanced X Server  New libc libraries 5.2.18 (Java ready!)

RPM 2.0 Features:

 complete C implementation (no longer requires perl)  now 10 times faster   
 RPMs are PGP signed, for authentication and security  backwards   
compatible with RPM 1.x  rebuild everything in a single command  handles   
configuration file modifications  failsafe upgrades  easier documentation   
searching

Even Easier Installation:

 "floppyless" install (requires dos to get started)  single floppy CD-Rom   
install option  also supports Red Hat's proven installation options  fine   
grained package selection in text install (with package descriptions)

GLINT: graphical package manager

 hierarchical package browsing  more than ten times faster than the   
previous 'lim' tool  easy to use system and package integrity checking

Enhanced Networking Capacities:

 includes Apache httpd preconfigured  graphical WWW Browser included   
 secure FTP daemon

XFree86 3.1.2:

 supports interactive mode configuration  868/968 S3 chips allowing   
nearly all Diamond Stealth to work
