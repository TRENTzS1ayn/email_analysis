The Amaya team is proud to announce the Amaya 8.0 release.

  Binaries (Linux, Windows NT/2000/XP /9x/ME), Debian and RPM 
  packages are available at:
            http://www.w3.org/Amaya/User/BinDist.html

  Source is available at:
            http://www.w3.org/Amaya/User/SourceDist.html

This version includes new important features:
  - new Accessibility features such as  menu access keys in Windows versions
  - All SVG transformation as well as several features of the SVG SMIL animation
    module in the OpenGL and WinGL versions
  - Support of the CSS property float.
  - Insertion of MathML and SVG elements within XML documents
  - Integration of the new 0.95 version of the expat XML parser.
  + several bug fixes
 
For more details see 
           http://www.w3.org/Amaya/User/New.html

 We thank all the users who helped us test the pre-release.
    Irene.
