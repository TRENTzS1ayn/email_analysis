You can make the index file readable:
1. Open index file (Help > Index)
2. Change to editor mode (Edit > Editor mode)
3. Open index.css file (Style > Open > Confirm)
4. Modify level1 and level2 classes
  padding-bottom and padding-top to 0px (from -4px)
5. Save index.css
