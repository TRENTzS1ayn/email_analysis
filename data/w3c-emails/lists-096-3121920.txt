Centering tables is still a problem with Amaya and style.
My short demo file has two tables, each bracketted with 
a div element.  The style rule for the div elemnt has both
text-align:center and margin:auto  to hedge bets -- either 
should do the job.  But the second one does not center.
The first DOES center because i have placed the old 
align=center attribute on the div .....   this is a workaround
and demonstrates that the rendering engine of amaya has the
capacity to center a dived table correctly.   But there is a
breakdown in the style property settings to rendering  that 
disables style centering of tables ....   headings and paragraphs
are centered ok inside divs   BUT NOT TABLES ....  This 
makes a lot of pages look bad and i havent been able to
construct a centered table in a new doc yet..... --
