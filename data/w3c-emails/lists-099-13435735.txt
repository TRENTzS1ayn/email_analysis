Dear Open Mobile Alliance,

  The XHTML Mobile Profile specification refers to the application/
vnd.wap.xhtml+xml MIME type which has so far not been registered.
Registration of MIME types is described in http://ietf.org/rfc/rfc2048;
since the type is in the vnd tree, registration should be as simple as
completing the form at <http://www.iana.org/cgi-bin/mediatypes.pl>. A
sample registration is included below.

  MIME media type name:
  
    application
  
  MIME subtype name:
  
    vnd.wap.xhtml+xml
  
  Required parameters:
  
    None
  
  Optional parameters:
  
    None
  
  Encoding considerations:
  
    The same encoding considerations as for the application/xml type as
    defined in RFC 3023 or its sucessor.
  
  Security considerations:
  
    The considerations as for the application/xhtml+xml type as defined
    in RFC 3234 or its sucessor.
  
  Interoperability considerations:
  
    The Open Mobile Alliance Ltd. Interoperability Working Group
    (IOP) assess interoperability for all Open Mobile Alliance Ltd.
    specifications.
  
  Published specification:
  
    The XHTML Mobile Profile specification [XHTMLMP].
  
  Applications which use this media type:
  
    ???
  
  Additional information:
  
    Magic number(s): n/a
    File extension(s): .xhtml, .xht
    Macintosh File Type Code(s): n/a
  
  Person & email address to contact for further information:
  
    ???
  
  Intended usage:
  
    LIMITED USE
  
  Author/Change controller:
  
    Open Mobile Alliance Ltd.
  
  [XHTMLMP]
