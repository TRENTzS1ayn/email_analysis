 I have made a small note which may be useful if you consider
developing a testbed using Amaya.

http://www.w3.org/TR/NOTE-expamaya

It explains the internal structure of the special versions of 
the Amaya client embedding a Java interpreter or offering ILU
interfaces.
The Java version is already available (a new version is due soon).
The ILU integration is nearly completed.

I would be interested in feedback if you use or plan to use one
of these experimental version.
