At the end of the interface and class definitions in the DOM
spec Draft 9-December-1997 there is after the closing brace "}"
always a ";".
While the javac compiler will not complain the ";" after the
brace "}" is not correct according to the Java Language Specification.

Occurs in:
http://www.w3.org/TR/WD-DOM/level-one-core-971209.html
Java Core API definitions

http://www.w3.org/TR/WD-DOM/level-one-xml
Java XML API definitions
