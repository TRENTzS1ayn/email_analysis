DOM XPath module last call issue a8 was about a description of the exception codes
returned in response to XPath expressions.  The resolution of this issue is that
descriptions of these codes are found in the normal place within the specification.

See the issues list for more complete information at 
http://www.w3.org/2002/03/DOM-Level-3-XPath-issues/all.html#a8

Please let us know if you are, or are not, satisfied with this resolution.
